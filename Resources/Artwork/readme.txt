The application icon set was created using the "Logo Design Studio":
http://www.summitsoft.com/

Icons were exported as PNG at the resolution of 1024x1024 and then rescaled
using this website: http://iconverticons.com/online/

Finally, a selected set of sizes was recombined into a single application
icon using this website: http://convertico.org/Multi_Image_to_one_icon/
